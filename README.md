# To run:
`python game.py`

# Usage:
Type commands until the game is over. A command is a line, ending when the user presses `ENTER`.

```
play sword
print
quit
```

# Commands:
`play <card>`: Play a card against the current enemy. Example: `play shield`
`<3`: Surprise~
`quit`: Quit the game
